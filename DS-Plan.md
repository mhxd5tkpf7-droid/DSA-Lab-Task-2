# Data Structure Planning Table -- Pharmacy Inventory System

  ------------------------------------------------------------------------
  Operation             Data Structure Used   Reason for Choosing the DS
  --------------------- --------------------- ----------------------------
  Add Medicine          Singly Linked List    Easy insertion at head, no
                                              shifting required.

  Display Inventory     Singly Linked List    Sequential traversal is
                                              simple and efficient.

  Search Medicine       Singly Linked List    Allows traversal through
                                              nodes to find records
                                              dynamically.

  Delete Medicine       Singly Linked List    Deleting a node is efficient
                                              without reorganizing the
                                              entire structure.

  Update Medicine       Singly Linked List    Updating fields in nodes is
  (Future Expansion)                          straightforward once
                                              located.
  ------------------------------------------------------------------------
